# ML_sustainable_concrete-
Abstract
This study focuses on early-stage concrete mix design, considering compressive strength, cost, embodied CO2 emissions, and energy consumption as primary and quantifiable sustainability indicators. Durability-related performance is not explicitly modeled but is indirectly reflected through strength-related mechanisms. An integrated surrogate–optimization framework is proposed, combining a physics-informed neural network (PINN) with a grid-based multi-objective evolutionary optimization (GMEO) algorithm for low-carbon concrete mix design. The PINN embeds the FIP hydration model into its learning process, enabling physically consistent prediction of compressive strength. Benchmarking against an artificial neural network and eight classical machine learning models demonstrates the superior predictive accuracy of the PINN, achieving R2 = 0.9815, RMSE = 2.98 MPa, and MAE = 2.01 MPa. SHAP-based interpretability analysis confirms that curing age, water–binder ratio, and binder composition are the dominant factors influencing strength, consistent with established material behavior. The GMEO algorithm is subsequently employed to minimize material cost, CO2 emissions, and embodied energy under strength constraints. Grid dominance and adaptive leader selection ensure stable convergence and well-distributed Pareto fronts, enabling the identification of balanced knee-point solutions. An interactive graphical user interface integrates the PINN–GMEO framework to support real-time computation, visualization, and decision-making. Experimental validation demonstrates strong generalization capability, with a relative prediction error of 0.84%. Overall, the framework effectively supports sustainable concrete mix designs by balancing strength and environmental impacts.
